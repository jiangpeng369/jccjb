# 金铲铲助手 - 项目说明

## 📱 项目简介

金铲铲助手是一款**合法的金铲铲之战辅助工具**，提供阵容推荐、概率计算、对战记录等功能。

**重要声明**：本应用不包含任何游戏自动化功能，不会修改游戏数据或进行违规操作。

---

## ✨ 功能特性

### 1. 阵容库
- 展示全网热门阵容
- 按热度、胜率、前四率排序
- 支持搜索阵容、英雄、羁绊
- 收藏喜欢的阵容
- 自适应屏幕宽度（1-4列网格）

### 2. 阵容详情
- 核心英雄展示（主C/主坦标识）
- 羁绊信息
- 运营思路（前期/中期/后期）
- 站位建议

### 3. 概率计算器
- **碰对手概率**：输入存活玩家，按概率排序
- **D牌概率**：计算追到三星的概率
- **装备概率**：估算获得目标装备的概率

### 4. 牌库计算器 ⭐
- 查看各费用卡牌的剩余数量
- 颜色编码显示卡池状态
- 追三星可行性分析
- 支持手动编辑卡牌数量

**卡池规则**：
| 费用 | 每张英雄数量 | 英雄种类数 |
|------|-------------|-----------|
| 1费  | 29张        | 13张      |
| 2费  | 22张        | 13张      |
| 3费  | 18张        | 13张      |
| 4费  | 12张        | 12张      |
| 5费  | 10张        | 8张       |

### 5. 对战记录
- 手动记录对战数据
- 统计各阵容胜率、前四率
- LP变化估算
- 数据本地存储

## 🚀 三种获取APK的方式

### 方式1：GitHub Actions自动构建（推荐）

无需安装任何软件，云端自动构建。

**步骤**：
1. 注册GitHub账号：https://github.com
2. 创建新仓库
3. 双击运行 `推送到GitHub_自动构建.bat`
4. 按提示输入GitHub仓库地址
5. 等待自动推送完成
6. 打开GitHub仓库 → Actions标签
7. 等待构建完成（约5-10分钟）
8. 在Artifacts中下载APK

详细步骤见：`GitHub自动构建指南.md`

### 方式2：本地一键打包

需要安装Android Studio。

**步骤**：
1. 安装Android Studio
2. 双击运行 `打包APK_双击运行.bat`
3. 选择Debug版本
4. 等待构建完成

详细步骤见：`打包APK_双击运行.bat` 和 `使用说明_请先读.txt`

### 方式3：Android Studio手动打包

**步骤**：
1. 安装Android Studio
2. File → Open → 选择TFT-Helper文件夹
3. 等待Gradle同步
4. Build → Build APK(s)

详细步骤见：`QUICK_START.md`

## 🛠️ 技术栈

- **语言**: Kotlin
- **最小SDK**: Android 7.0 (API 24)
- **目标SDK**: Android 14 (API 34)
- **UI框架**: Android原生 + Material Design
- **网络**: OkHttp + Retrofit
- **图片加载**: Glide
- **数据解析**: Gson
- **协程**: Kotlin Coroutines
