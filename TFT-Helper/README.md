# 金铲铲之战助手

一款合法的金铲铲之战辅助工具，提供阵容推荐、概率计算、对战记录等功能。

## ⚠️ 重要声明

**本应用不包含任何游戏自动化功能，不会修改游戏数据或进行任何违规操作。**

本应用仅提供：
- 公开阵容数据展示
- 概率计算工具（纯数学计算）
- 对战记录（手动输入）

## 功能特性

### 1. 阵容库
- 展示全网热门阵容
- 支持按热度、胜率、前四率排序
- 阵容收藏功能
- 搜索功能
- **自适应屏幕宽度，自动计算网格列数**

### 2. 阵容详情
- 核心英雄展示
- 羁绊信息
- 运营思路（前期/中期/后期）
- 站位建议

### 3. 概率计算器
- **碰对手概率计算** - 输入存活玩家信息，按概率排序
- D牌概率计算 - 计算追到三星的概率
- 装备概率估算

### 4. 牌库计算器 ⭐新增
- 查看各费用卡牌的剩余数量
- 追三星可行性分析
- 卡池分布可视化
- 支持手动编辑卡牌数量

### 5. 对战记录
- 手动记录对战数据
- 统计各阵容胜率、前四率
- LP变化估算
- 数据本地存储

## 概率计算原理

### 碰对手概率
在8人局中，每个玩家每回合会匹配一个随机对手（不会匹配自己）。
- 基础概率 = 1 / (存活玩家数 - 1)
- 考虑上回合对手，降低连续相遇概率

### D牌概率
- 根据等级确定各费卡刷新概率
- 根据卡池剩余数量计算特定英雄出现概率
- 使用泊松分布估算追到三星的概率

## 使用说明

### 查看阵容
1. 打开应用，进入阵容库
2. 选择排序方式（热度/胜率/前四率）
3. 点击阵容卡片查看详情

### 概率计算
1. 点击右下角计算器图标
2. 选择计算类型
3. 输入相关参数
4. 查看计算结果

### 记录对战
1. 点击左下角记录图标
2. 点击添加按钮
3. 填写对局信息
4. 保存记录
